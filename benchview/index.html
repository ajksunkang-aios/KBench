Benchmark Catalog

The SWE-bench for Operating Systems · github.com/ajksunkang-aios/KBench

KBench is a multi-vertical benchmark for OS-kernel agent engineering. The catalog below lists each test suite by category, with its size, format, and provenance. Datasets are imported from the cited sources; the harness measures accuracy and cost (tokens / tool-calls / time) under a controlled A/B (baseline grep/glob/read vs +KGraph).

Evaluation

Each task runs an agent (model + tools) under two arms — A-baseline (grep/glob/read) vs B-kgraph (baseline + KGraph's 13 MCP tools) — instrumented for accuracy (recall/precision) and cost (tokens / tool-calls / wall-time). Results render as an A-vs-B report per vertical.

Full design & methodology: docs/DESIGN.md · Retrieval v1 quick-run result: KGraph 2.5× fewer tokens / 5× fewer calls / 6× faster at equal accuracy.
